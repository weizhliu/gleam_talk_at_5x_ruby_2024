# Lets Gleam
介紹 Gleam ，2024/4/23 默默會分享用專案


## 9大特點
1. develop ux
2. functional
3. simple syntax
4. static typing
5. result type
6. pattern matching
7. named arguments
8. compile to JavaScript
9. compile to Erlang


## 執行特點範例
```sh
gleam run -m lets_gleam 2
```

## lustre: start a development server
```sh
gleam run -m lustre/dev start
```
